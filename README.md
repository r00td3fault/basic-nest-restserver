# Desarrrollo

1. Clonar el repositorio
2. Ejecutar 
```
yarn install
```
3. Tener nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar base de datos
```
docker-compose up -d 
```

5. Insertar data inicial
```
http://localhost:3000/api/v2/seed
```

* MongoDB
